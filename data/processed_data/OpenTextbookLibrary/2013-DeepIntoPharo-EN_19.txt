When there is a need to access a portion of a path, the message path Segments returns the full name cut into path elements, as strings. Remember that from a design point of view, strings are considered as dead objects, so it is often better to deal with the real objects for example using the path message. pf path Segments ('Users' 'ducasse' 'Pharo' 'Pharo Harvesting Fixes' ' ' 'package -cache' ' Asm Jit -Igor Stasenko. .mcz') pf path Path 'Users' 'ducasse' 'Pharo' 'Pharo Harvesting Fixes' ' ' 'package - cache' 'Asm Jit -Igor Stasenko. .mcz' Sizes.
