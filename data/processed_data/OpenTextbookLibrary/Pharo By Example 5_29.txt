Typenowin the top left search box, click Search (or just press the Enterkey). You should see a list of results similar to the one in Figure - . The Finder will display a list of all the method names that contain the substring now . To scroll to nowitself, move the cursor to the list and type n ; this type-ahead trick works in all scrolling windows. Expanding the now item shows you the classes that implement a method with this name. Selecting any one of them will display the source code for the implementation in the code pane on the bottom.
