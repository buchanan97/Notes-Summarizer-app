Type the expression 4into the workspace. Now do it with the keyboard shortcut. Do not be surprised if you saw nothing happen! What you just did is send the message with argument 4to the number . Normally the result will have been computed and returned to you, but since the workspace did not know what to do with this answer, it simply threw the answer away. If you want to see the result, you should print it instead. print it actually compiles the expression, executes it, sends the message print String to the result, and displays the resulting string.
