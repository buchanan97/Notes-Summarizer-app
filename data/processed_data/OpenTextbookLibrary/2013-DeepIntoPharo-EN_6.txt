A zero con guration is a script that automatically downloads everything you need to get started. Since version , Pharo also supports a way to de ne and handle command line arguments. This chapter shows how to get the zeroconf scripts for Pharo as well as how you can pass arguments to the environment from the command-line. Getting the VM and the Image First here is a way to download a zero con guration script to download the latest Pharo image and vm. wget get.pharo.org vm If you do not have wget installed you can use curl-Linstead. Zero Con guration Scripts and Command-Line Handlers To execute the script that we just downloaded, you should change its permissions using chmod a x or invoke it via bash as follows. Con gurations. There is a plethora of con gurations available. The URL for each script can be easily built from an image version and a vm following the expression: get.pharo.org IMAGE VM Possible values for IMAGE are: stable alpha Possible values for VM are: vm vm S vm Latest vm SLatest Of course, one can just download an image as well get.pharo.org IMAGE or just the VM get.pharo.org VM Looking at the help. Now let s have a look at the script help. bash vm --help The help says that the vm command downloads the current virtual machine and puts it into the pharo-vm folder.
