Open Source At this point, you have come across many references to terms like free, free to edit, and open source throughout the text. In fact, all of the elements in our example LAMP are free, open source solutions. Open source means the provider of the software allows the end user access to the actual code of their software, allowing the end user to make changes anywhere in the program. This differs from traditional software where you own a copy or license to use the program, but cannot extend or change elements of the program beyond what the developer allows.
