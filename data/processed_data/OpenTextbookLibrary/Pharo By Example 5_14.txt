If you double click on the VM, it looks for an image file in the default location. If you double click on an .image file, it tries to find the nearest VM to launch it with. If you have multiple VMs installed on your machine, the operating system may no longer be able to guess the right one. In this case, it is safer to specify exactly which ones you meant to launch, either by dragging and dropping the image file onto the VM, or specifying the image on the command line (see the next section).
